# Frontend

React + deck.gl interface for the CHI drone visual exposure prototype.

## Development

```powershell
cd D:\CHI\frontend
npm install
npm run dev
```

The app reads the backend base URL from `VITE_API_BASE_URL`. The default is:

```text
http://127.0.0.1:8011
```

## Tests

```powershell
cd D:\CHI\frontend
npm test
```
